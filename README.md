*Anti virus Software*
A Swing java application for scanning and getting rid of malware

The purpose of this application is to get rid of viruses or forms of malware or at best quarantine it.
The application has several functions:
1.Scanning 
2.Quarantining
3.Deleting 
4.Restoring 
5.File handling(saving information)
6.and Logging information

It includes a setting feature for the scanner ,quarantine and application in general
Options on virus scan: full and custom
Quarantine Manager redirects the files path into a hidden quarantine folder with the virus having its own extension .quarantine
Option on deleting or restoring to original place

####Installation####
1.Open the project in Netbeans or IntelliJ
2.Make sure you got JDK17+ installed
3.Build and Run 

***Use***
1.Open application
2.Press enter
3.Go to scanner 
4.Select a scanner method 
NOTE:Preferably use a custom scanner,and test on a dummy file ,to avoid scanning through your whole computer and quarantining important files 
5.Go to quarantine zone 
6.Select what to do with the files in the BlackList


---Dependencies---
JDK17+
Swing GUI
IDE supporting JAVA
